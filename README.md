# md.md
A Markdown Cheatsheet 

### Headings

```
# H1
## H2
### H3
#### H4  
##### H5
###### H6
```

### Highlighting

```
**Bold**
```

```
*Italic*
```

```
~~Strikethrough~~
```

```
> Block quote
```

### Lists

**Unordered**

```
* Item 1
* Item 2
  * Item 2a
  * Item 2b
    * Item 2c
    * Item 2d
```

**Ordered**

```
1. Item 1
1. Item 2
  1. Item 2a
  1. Item 2b
1. Item 3
  1. Item 3a
    1. Item 3b
    1. Item 3c
```

### Syntax Highlighting

**Code block**
```
```Javascript
var name = "Jonathan";```
```
*Adding Javascript after the backticks enables syntax highlighting.*

**Inline code**

```
Sometimes you might need to include a `short snippet` in the middle of a sentence.
```

### Tables

```
Title 1 | Title 2
---------- | ----------
Row 1, cell 1 | Row 1, cell 2
Row 2, cell 1 | Row 2, cell 2
```

### Task Lists

```
- [x] Complete tasks
- [ ] Incomplete task
```
